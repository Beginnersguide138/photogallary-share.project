# photogallary-share.project

## About This Repository

The collection of works by Eisuke Izawa. These are created during 2019 - 2023.

## Works

### De Stijil -- The Style of Photograph

#### Explanation

A work attempted to describe the perspective of a photographer and the recognition of him that lies on his mind, which occurred on an occation.
This work uses CSS Grid, Animation and Text-Shadow for special expression of above thing.

### Slides of F/A-18 Hornet

#### Explanation

A work attempted to make new UI. In this work, users must click button to move to next paragraph or screen and text passages to go forward.
jQuery methods are used repeatedly to make slide effects.

### Horizontal Parallax

#### Explanation

A work attempted to introduce horizontal parallax & new effect style.
Unlike common callendar display style, this work implements an easy & simple display style, with x-axis being month & y-axis being date.
jQuery is used to create such vidual effect.
